Genetic Cellular Automaton , 2020 Feb 05

The Genetic Cellular Automaton is a cellular automaton which uses genetic algorithms to reach a perfect/controlled state from a default state that is in variance.

Star Ghost is a simple 1 and 2 player game designed in Unity, inspired partly by StarFox. It is designed to re-use the avionics simulation of Electroshka, as well as having a “ground control” type player.

Dial Duel is a game about rapid hacking. It is prototyped through a terminal and communication is done through a secure tunnel. Bittish is used as the language very naturally in order to issue commands/actions.

Tetris Clone , 2010 Aug 11

This was developed in 2010 as my last C++/Allegro project before I began to study Simple DirectMedia Layer. It is a simple Tetris clone which features a score and various blocks.

Meteoroids , 2010 Aug 12

This was my first game developed using C++ and Simple DirectMedia Layer back in 2010. It is largely based on Asteroids (but not quite the same). It is programmed for the GP2X-Wiz.

Lost Labyrinth , 2015 Jan 15

This is a game developed with the help of a small team under 48 hours for the Global Game Jam competition. It is a somewhat of a hybrid between a top-down (or isometric) game and a 2D side scrolling maze. The map keeps track of the player coordinates within the maze, and based on that information, the 2D side scrolling world is rendered.

Roommate Tools , 2016 Apr 01

The four of us, Matt B., Raymond G., William V., and Khoi D., set out to create a useful tool for university students. We ended up developing Roommate Tools, a practical application whose primary concern is to allow students to find potential roommates.

Arithmetic Logic Unit , 2016 Apr 25

This project is a traditional arithmetic logic unit (that is, the hardware inside the computer which runs arithmetic and logical operations) modeled in MIPS assembly. The assembly simulation software used is MARS.

One Tough Program , 2020 Feb 05

One Tough Puzzle is the name of a classic jigsaw puzzle that I received five to ten years ago. I had heard of a similar puzzle without a pattern, and reckoning the difficulty, I had never actually bothered. But we live in the information age, and deciding to brush up on my C skills, I wrote a program to solve it. This could be genericized to solve any jigsaw puzzle (if the puzzles joints are enumerable).
